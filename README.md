## Install & Run

To **clone** this repository, you'll need to have **git** installed on your computer.<br>
From your command line:

```bash

git clone https://github.com/CM0use/README-TEMPLATE.git
cd README-TEMPLATE

```

Run on **macOS & Linux**:

```bash

nano README.md

```

Run on **Windows**:

```cmd

README.md

```
